About TAMinations

Back

TAMinations are the Square Dance animations of the Tam Twirlers Square Dance Club. Here you will find animations for calls from Mainstream through C3A. Most calls have several animations from different starting positions. TAMinations are written and maintained by Brad Christie.

This modified iOS web app version was created by M. C. DeMarco in March 2012. To use the app offline, open it first in Safari for iOS and click Safari's "Add to Home Page" button. You must be online the first time you run the app from its Home Page link.

Known Problems

  • Animations do not display on Android devices when using the default browser. This is because the browser does not support SVG, which Taminations uses. To fix this, install and use a different browser such as Opera or Firefox. The next version of Android (4.0 aka Ice Cream Sandwich) supposedly does support SVG.
  • Manually dragging the slider does not change the dancer's position. To be fixed.
